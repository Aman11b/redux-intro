# 💰 Redux Banking App

A simple banking application built with **React** and **Redux Toolkit** to learn modern Redux concepts and global state management.

---

## ✨ Features

- 💵 Deposit and withdraw money
- 🏦 Request and pay loans
- 🌍 Currency conversion using async Redux Thunk
- 🗂️ Global state management with Redux Toolkit
- ⚛️ React-Redux hooks (`useDispatch`, `useSelector`)

---

## 🛠️ Tech Stack

- ⚛️ React
- 🧰 Redux Toolkit
- 🔄 React Redux
- 📜 JavaScript

---

## 📚 Redux Concepts Covered

- 🏪 Store Configuration
- 🧩 Slices & Reducers
- 🎯 Action Creators
- ⏳ Async Thunks
- 🔌 Middleware
- 🌐 Global State Management

---

## 🎯 Learning Purpose

This project was built to practice:

- Modern Redux Toolkit workflow
- Async actions with Thunk
- Scalable state management in React applications
- Redux architecture and best practices
